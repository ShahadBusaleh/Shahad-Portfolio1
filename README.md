# Shahad Busaleh — Portfolio

Personal project portfolio site for **Shahad Busaleh**, Full Stack Developer.

## Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS v4
- Bilingual EN / AR with RTL support

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Deploy to [Vercel](https://vercel.com) or any static/Node host:

```bash
npm run build
npm start
```

## Brand

Colors match the LinkedIn profile branding:

- Teal: `#5a7d85`
- Cream: `#e8dec9`

## Structure

- `src/data/` — profile, projects, skills content
- `src/components/` — UI sections
- `src/lib/i18n.ts` — navigation and UI strings

Update project links and contact info in `src/data/profile.ts` and `src/data/projects.ts`.
